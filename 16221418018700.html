5. 分支循环

分之结构

if...else...

if(a) {
    xxxxx;
}

// 二选一
if(a) {
    xxxxx;
} else {
    xxxxx;
}

// 多选一
if(a) {
    xxxxx;
} else if(b) {
    xxxxx;
} else {
    xxxxx;
}

switch...case...

  • switch 中的表达式的类型只能是以下 6 种: byte short char int 枚举 String
  • case 后只能写常量, 不能是范围
  • 当匹配到一个分支, 后面的分支不需要判断直接执行, 如果不想执行需要添加 break
switch(a) { // a 的类型只能是以下6种: byte short char int 枚举 String
    case b: // a == b
        xxxxx;
        break; // 不加 break 会不进行判断直接继续向下执行 case c 和 default 的代码
    case c: // a == c
        xxxxx;
    default: // 位置可任意
        xxxxx;
}

循环结构

① 初始化条件
② 循环条件
③ 迭代部分
④ 循环体

for

for ( ①; ②; ③ ) {
    ④;
}

while

①
while ( ② ) { // 满足进入
    ④;
    ③;
}

do...while

①
do {
    ④;
    ③;
} while ( ② ); // 满足再次进入
