equals

==: 比较的是两个变量在栈中的内容，如果变量时引用类型，栈中保存的时对象地址
equals: Object 类中默认使用的是 ==，如果要比较的是引用类型变量指向的堆内存中的对象，需要重写该方法实现具体的比较逻辑

public boolean equals(Object o) {
    if (this == o) return true;
    if (o == null || getClass() != o.getClass()) return false;
